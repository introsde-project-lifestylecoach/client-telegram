# client-telegram
A telegram bot client for lifestyle coach

## Procedures

### Registration
1. User exist
2. Firstname (taken from Telegram)
3. Surname
4. Sex (m or f)
5. Birthdate
6. Current height
7. Current weight
8. Current waist
9. Current hip

#### /seeprofile
Return firstname, surname, height, birthday, last weight, last waist, last hip

#### /measure
1. Select type -> Show last 15 measures of that type
2. Insert new value 

#### /Goal
1. Show goals
  a. New Goal
    i. Title
    ii. Description
    iii. Type
    iv. Condition
    v. Value    
  b. Update goal
  c. Delete goal
  
#### /adaptor
Calculate BMI

### /info
Show the commands of the bot
